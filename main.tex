\section{Instantaneous Rate of Change: The Derivative}

Difference quotient
\[ \frac{ f(x + \Delta x) - f(x) } {\Delta x} \]



Limit
\[ 	
	\biggl( \lim_{x \to a}f(x)=L \biggr) \Rightarrow
	\biggl(
		\left( \forall \epsilon > 0 \right) 
		\left( \exists \delta > 0 \right) 
		\left( 0 < \left|x - a \right| < \delta \rightarrow 
			   \left| f(x) - L \right| < \epsilon \right)
	\biggr)
\]
